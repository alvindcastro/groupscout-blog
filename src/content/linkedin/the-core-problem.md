Building permits are public. But finding them is a full-time job.

Most companies have someone manually scouring city websites, downloading PDFs, and trying to figure out which project is worth a phone call.

I built GroupScout to fix that.

The goal: **Go from raw data to a qualified lead in seconds.**

1. **Collect:** Automate the poll for new building permits across multiple sources.
2. **Score:** Use an LLM (Claude/Ollama) to estimate crew size and project duration.
3. **Notify:** Send high-priority leads directly to Slack.

Stop searching. Start scouted.
