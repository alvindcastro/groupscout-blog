Group Scout has no dashboard. No login. No UI.

And that’s by design.

When I started building this for my wife, the obvious "developer" answer was to build a web app. But the last thing a busy sales manager needs is another tab to check and another password to remember.

Her job is to make calls, not to manage a CRM-lite.

So the entire "UI" is a Slack message that hits her phone on Monday morning.

Using Slack’s Block Kit, I can send rich cards with priority emojis (🔥 for high priority), direct links to source PDFs, and Claude’s outreach suggestions.

The lesson: Sometimes the best UI is the one your user is already using.
