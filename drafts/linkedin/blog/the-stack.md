The best tech stack for a side project is the one that gets the job done without being a maintenance burden.

For Group Scout, that means:
1. Go: A fast and reliable backend for data collection.
2. PostgreSQL + pgvector: A robust database for storage and similarity searches.
3. n8n: A flexible workflow tool for orchestrating API calls and notifications.
4. Claude (Anthropic): A context layer for lead scoring and enrichment.
5. Slack: The primary interface for lead delivery.

I chose this stack to focus on the 'signal'—finding leads that hotels want. The stack serves the data, not the other way around.
