Privacy and cost are the two biggest concerns for developers using AI.

For Group Scout, I am experimenting with self-hosting LLMs for local development. I use Ollama to run models like Llama 3 on my local machine.

Three benefits:
1. Zero cost. I can iterate on prompts for thousands of leads without an API bill.
2. Data privacy. The permit text stays on my machine during development.
3. Fast iteration. Ollama starts models in seconds, and I can switch between Llama, Mistral, and Phi with one command.

For production, I still use Claude Haiku for its accuracy and structured JSON output. But for development, Ollama is my primary tool.

Self-hosting an LLM allows for more experimentation and fewer API limits.
